# CoreHives Project Structure

## Folders
- `components/` — Reusable UI components
- `components/layout/` — Layout components (Header, Footer)
- `components/animations/` — Animation-related components
- `pages/` — Route-level pages
- `assets/` — Static assets (images, icons, sound)
- `utils/` — Utility/helper functions
- `hooks/` — Custom React hooks
- `services/` — API and service logic

## Scripts
- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run preview` — Preview production build
- `npm run lint` — Lint code

## Notes
- UI code is untouched and 10/10 as requested.
- Scaffolded for maintainability and scalability.
- Add new utilities to `utils/`, hooks to `hooks/`, and API logic to `services/`.
